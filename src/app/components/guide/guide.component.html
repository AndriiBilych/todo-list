Description

This website is a todo type web app made for training purposes by Andrii Bilych. It was created as a way to practice my skills and improve my knowledge on Angular/TypeScript/HTML/CSS. The website contains only front-end part, therefore all data will be lost upon reloading or choosing another board. Todo list loads with two boards filled with fake data received from an http request to json file, as if receiving a response from back-end. Project is inspired and modeled after Trello boards.

Features

  • Create/Delete/Modify tasks
  • Create/Delete lists
  • Change list name by pressing its title
  • Reorder tasks by dragging them inside list
  • Move tasks from list to list
  • Switch boards
  • Create multiple tasks by pressing enter when task creation form is opened
  • Cancel task creation by pressing ESC
